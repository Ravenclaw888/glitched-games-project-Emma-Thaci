How to play 
1. Controls
-   Up arrow - Jump
-   Down Arrow - Super Jump
-   Right Arrow - Right
-   Left Arrow - Left
-   A Button / Space / "Q" - End Hint(You will be propmted)
-   B Button / "E" - Gives Hint when 1 minute cooldown timer at the top is done
-   Move on from each level by overlapping with the green and orange blocks
2. Backstory
- This is inspired off of old arcade games, many of which are slowly degrading due to the time since they were created. Plus, the ending isa bit horror-inspired based of science fiction films about computer sentience.
3. Time it takes to play
- 10-15 minutes on first run through.
4. Things to note
- Makecode Arcade is known to have slow download on Github, so refreshing 2-3 times before playing allows for the best user experience.
